我的博客：https://tony031218.github.io/

在文件夹外面的是我在考前整理的模板

- __`JLOI2020_rp++.cpp`__：部分省选模板（摸了_(:з」∠)_

- __`Mathematics.cpp`__：数论模板
- __`StringAlgorithms.cpp`__：字符串模板
